Setting up your development environment

  • Choose an IDE

    • IDE stands for Integrated Development Environment. It's a program where you'll code and do your magic!.
    There're many IDEs, you can choose the one that you feel most comfortable using.


    If you don't have any preference, we recommend that you use Visual Studio Code. Download it here.
    One of the best things about VS Code is that it offers many extensions that can make your life easier.

  • Install git

    • We need a way to track code changes, for that we use a version control system called git.
    Follow the steps to install it depending on your OS here.

  • Install Node.js

    • Node.js is an open-source and cross-platform JavaScript runtime environment.
    Check out how to install it depending on your OS here.

  • Github account

    • You'll need a Github account if you don't already have one. This will allow you to have a version controll of your projects.
    In case you have a personal account, you can use that one. If you don't. sign up here.

  • Heroku account

    • Also, for this MooveCamp you'll need to have a Heroku account for deploying your final challenge.
    If you don't have an account yet, you can register here.
