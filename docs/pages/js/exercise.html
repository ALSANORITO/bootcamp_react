JavaScript exercises

Given an array of numbers, create a function that returns:


  1. the element with the lowest value.
  2. the element with the highest value.
  3. the sum of all elements (and if they are not all numbers display an error message).

Create an arrow function that:


  1. receives three parameters: name, age and hobbies.
  2. and returns a personalized formatted text as a greeting.
  3. Example:
    >> printGreeting("Maria", 36, ["biking", "painting", "snorkelling"])
    >> "Hello! I'm Maria, and Im 36 years old. In my free time I like biking, painting and snorkelling."
